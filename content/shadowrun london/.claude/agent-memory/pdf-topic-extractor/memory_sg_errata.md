## Other Notable Errata (for future reference)

### Spell Range Changes
Catalog T(A) | Chaff LOS(A) | Fashion T(A) | Increase/Decrease Noise LOS(A) | Insecticide [insect spirit] LOS(A) | Mass [Sense] Removal LOS(A) | Spatial Sense T(A) | Spatial Sense Extended T(A) | Stench LOS(A)

### Notable Rule Changes
- Ambidexterity spell: Duration changed from S to I
- Physical Camouflage spell: Type changed from M to P
- Net Bind: Now Physical, Area
- Interference spell: Now Physical, Area (not Environmental)
- [Element] Wall: Size adjustment requires Spell Shaping metamagic; damage is "Damage Value equal to spell's Force" not Drain
- Destroy [Vehicle]: Type changed from M to P
- Insecticide: Type changed from P to M
- Endowment spirit power: Targets = Magic (not twice Magic)
- Commanding Voice adept power: Renamed Authoritative Tone, 0.5 PP/level (max 3)
- Inertia Strike: Adds (Magic x 0.5) to DV only to determine knockdown — does not change actual DV
- Digital Celerity: –1 Perception per initiate grade (not per level of Nimble Fingers)
- Beast's Way: Remove the last sentence granting one additional power
